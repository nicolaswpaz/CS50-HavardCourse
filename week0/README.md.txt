# CS50 - Semana 1: Projeto de Scratch - Jogo do Labirinto

Este projeto foi desenvolvido como parte da **Semana 1 do CS50: Introdução à Ciência da Computação**, curso ministrado por David J. Malan na Universidade de Harvard.

Na primeira semana, aprendemos os conceitos fundamentais de lógica computacional, algoritmos e pensamento computacional sem necessidade de escrever código em linguagens tradicionais como C ou Python. Em vez disso, utilizamos o [Scratch](https://scratch.mit.edu/), uma linguagem de programação visual desenvolvida pelo MIT, que permite criar projetos interativos arrastando blocos de código.

## Objetivo do Projeto

O objetivo deste projeto era criar um jogo interativo no Scratch, aplicando os princípios de:

- **Eventos e respostas** (por exemplo, pressionar teclas)
- **Laços de repetição**
- **Condições**
- **Detecção de colisão**
- **Variáveis e pontuação**

## Descrição do Jogo

**Nome do Jogo:** Labirinto do Gato Estelar

Neste jogo, você controla um gato aventureiro que precisa escapar de um labirinto, coletar estrelas e finalmente chegar ao seu destino: o planeta dos gatos.

### Como Jogar

- **Movimentação:** Use as teclas de seta para mover o gato pelo labirinto.
- **Objetivo:** 
  - Colete todas as estrelas espalhadas pelos caminhos.
  - Evite encostar no cachorro que patrulha o labirinto.
  - Após coletar todas as estrelas, encontre o caminho até o planeta dos gatos para vencer.
  
### Mecânicas Principais

✅ **Controle com setas**  
✅ **Detecção de colisão com obstáculos e inimigos**  
✅ **Coleta de itens (estrelas)**  
✅ **Tela de vitória ao completar os objetivos**

## Sobre o CS50

O [CS50](https://cs50.harvard.edu/) é um dos cursos de introdução à ciência da computação mais reconhecidos do mundo, com foco em desenvolver o pensamento computacional e apresentar várias linguagens de programação ao longo das semanas. Na primeira semana, a principal proposta é explorar a criatividade e os fundamentos da lógica usando Scratch.

## Como Executar

1. Acesse o projeto no [Scratch](https://scratch.mit.edu/projects/1179940352).
2. Clique em “Ver Dentro” para visualizar os blocos de programação.
3. Clique em “Bandeira Verde” para iniciar o jogo.
4. Divirta-se explorando o labirinto!
